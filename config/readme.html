Demo

This dashboard is meant to serve as a basic demonstration of the kind of queries and visualizations you can make with some of the data you collect with the extension.

It's currently under development, so there might be a bug or missing feature or two.

How to query the data

The data in this dashboard primarly leverages Quickwit. Quickwit supports a Lucene-like query language (and Elasticsearch-search compatible API) which you can learn more about here.

At the top of the dashboard you can also filter by log and trace attributes, which will automatically apply to the related dashboard visualizations.
